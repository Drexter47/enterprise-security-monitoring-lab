# Resume Bullets

- Built an Enterprise Security Monitoring & Incident Response Lab using Ubuntu Server and Windows 11 to implement SSH hardening, host firewall policies, audit logging, and brute-force protection.
- Investigated Linux and Windows security events, including SSH brute-force attempts, local user creation, and administrator group modification, and documented findings in structured incident reports.
- Developed a Python security log analyzer that parses recent SSH authentication activity, calculates a risk score, identifies top offending IPs, and generates Markdown/HTML security summaries.
- Mapped simulated attack scenarios to the MITRE ATT&CK framework to standardize detection and incident response documentation.
