# Detection Index

This folder contains detection playbooks for the Enterprise Security Monitoring & Incident Response Lab.

## Detection Playbooks

- [SSH Brute Force](ssh-bruteforce.md)
- [Windows Local User Creation](windows-user-created.md)
- [Administrator Group Modification](administrator-added.md)
- [SSH Configuration Modification](ssh-config-modification.md)
- [Failed Windows Login](failed-windows-login.md)
- [auditd File Change Monitoring](auditd-file-change.md)

## Purpose

Each detection playbook documents:

- What activity is being detected
- Why it matters
- How to investigate it
- What evidence to collect
- How to respond
- How it maps to MITRE ATT&CK

These playbooks are written to reflect how a SOC analyst or security engineer would handle alerts in an enterprise environment.
